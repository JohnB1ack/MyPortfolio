The Academic Challenge

During my statistical programming class at Miami University, I was assigned a complex data analysis problem involving a large dataset of movies with extensive metrics. The challenge wasn't just about the individual movies - my professor wanted to analyze the metrics of the genres the movies belonged to, rather than the specific films themselves.

This required transforming raw movie data where much of the information was stored in JSON format, making extraction more complex. Additionally, movies belonged to multiple genres, requiring careful data restructuring to enable meaningful analysis.

My Solution

I developed a comprehensive SAS macro solution that could extract genres from each movie and restructure them into separate rows for analysis. The solution allows users to input any genre and receive detailed statistics and visualizations for that movie genre, creating a flexible analytical tool.

The approach involved creating a macro that users could call with specific genre parameters, which would then return comprehensive statistics and visualizations. This demonstrated early problem-solving skills that would later become central to my citizen development approach.

Technical Implementation

The solution required both SAS and SQL integration to achieve the desired analytical capabilities. I used SQL statements to relate the extracted genre data back to the original dataset, enabling complex analysis like runtime correlations.

Results & Visualizations

The final solution produced comprehensive visualizations showing rankings and popularity metrics for selected genres. These charts demonstrate the analytical capabilities that would later inform my approach to business intelligence and data visualization in enterprise environments.

Foundation for Future Development

This academic project established the analytical foundation and problem-solving methodology that drives my current approach to citizen development. The patience required to work with complex data structures and the mission to make data accessible and meaningful to users became core principles in my professional work.

The skills developed here - data extraction, transformation, macro development, and user-focused design - directly translate to the enterprise automation solutions I create today. This project taught me that the most important aspect isn't just having technical answers, but understanding the problem deeply enough that the right solution reveals itself.
