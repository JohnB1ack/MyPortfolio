Data Cleaning
in SQL & SAS

I was assigned this problem in my statistical programming class, where we had a large amount of data about individual movies and a lot of metrics on those movies. However my professor wanted to look further into the metrics of the genres the movies belonged to, rather than the specific movies. Knowing this started to look into how I was going to clean and extract this raw movie data.

This is an example of the CSV file that I was working with. As you can see one of the main issues was that much of the data was in JSON format making it slightly more difficult to extract. Another problem is that the movies have multiple genres they belong to. My professor wanted to be able to look at rating metrics of whatever genre in a simple way. My solution was to create a macro to call where the user can input any genre and the code will return the statistics on that movie genre.

This code extracts the genres from each movie and puts them into seperate rows. Click on the image of the code above to view all of the code. The resulting code allows users to specify a list of genres that they'd like to compare, and then produces visualizations for the user. In the below example visuals the genres Adventure, Action, and Foreign are used in the macro.
